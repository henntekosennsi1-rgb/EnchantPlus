# EnchantPlus - カスタムエンチャントシステム

19種類のカスタムエンチャント、カスタム金床GUI、村人取引システムを統合した本格的なエンチャントプラグインです。

---

## 特徴

- **19種類のカスタムエンチャント** - 基本・採掘・戦闘・特殊能力系
- **カスタム金床GUI** - 金床右クリックでGUI起動
- **村人取引システム** - エンチャント商人を召喚
- **統合版対応** - 15種類がGeyser/Floodgate対応
- **最大レベル25** - 全エンチャントがLv25まで
- **エンチャント合成** - Lv5 + Lv5 = Lv6

---

## エンチャント一覧

### 基本系（6種類）
| エンチャント | 効果 | 統合版 |
|-------------|------|--------|
| 移動速度上昇 | +2%/Lv | ✅ |
| ノックバック耐性 | +10%/Lv | ✅ |
| 段差上限解除 | +0.1ブロック/Lv | ❌ |
| ジャンプ力上昇 | +0.1ブロック/Lv | ✅ |
| ブロック探索範囲 | +0.5ブロック/Lv | ❌ |
| 経験値取得倍率 | +5%/Lv | ✅ |

### 採掘・収集系（3種類）
- 採掘速度上昇、幸運の祝福、アイテム修理

### 戦闘・防御系（5種類）
- 炎耐性、再生能力、吸収シールド、防御力強化、棘の鎧強化

### 特殊能力系（5種類）
- 水中呼吸、暗視、透明化、満腹度維持、落下ダメージ無効

## コマンド

| コマンド | 説明 | 権限 |
|---------|------|------|
| `/enchantplus` | GUI起動 | `enchantplus.gui.use` |
| `/enchant list` | 一覧表示 | `enchantplus.command.list` |
| `/enchant give <player> <enchant> <level>` | 付与 | `enchantplus.command.give` |
| `/enchantvillager` | 商人召喚 | `enchantplus.villager.summon` |

---

## 権限プリセット

| 権限 | 対象 |
|------|------|
| `enchantplus.player` | 一般プレイヤー |
| `enchantplus.moderator` | モデレーター |
| `enchantplus.admin` | 管理者 |
